Colors explained in CSS

  1. Hue

    From left to right on the color wheel

  2. Tint

    Adding white to a color creates a tint, which lightens the color.from right to left.

  3. shade

    Adding black to a color creates a shade, which darkens the color.From top top bottom.

  4. tone

    Adding gray to a color creates a tone, which reduces the saturation of the color.From right top right corner to left down corner
